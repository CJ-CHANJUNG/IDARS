## ⚡ 3분 빠른 시작

### 1단계: 설치 (30초)
```bash
pip install PyMuPDF
```

### 2단계: 설정 (1분)
`posco_parser.py` 파일을 메모장으로 열기
→ 맨 위 부분만 수정:

```python
# 🔧 사용자 설정 (여기만 수정하세요!)
INPUT_FOLDER = r"D:\내PDF폴더"  # ← 여기에 경로 입력!
```

**경로 입력 방법:**
1. 파일 탐색기에서 PDF 폴더 열기
2. 주소창 클릭 → Ctrl+C
3. `r"` 다음에 Ctrl+V
4. 저장 (Ctrl+S)

### 3단계: 실행 (10초)
```bash
python posco_parser.py
```

### 4단계: 확인
`parsed_results` 폴더에 JSON 파일 생성됨! 🎉

## 🐛 자주 하는 실수

### ❌ 실수 1: 경로에 r 안 붙임
```python
INPUT_FOLDER = "D:\내폴더"  # 잘못됨!
```
**해결**: 
```python
INPUT_FOLDER = r"D:\내폴더"  # r 붙이기!
```

### ❌ 실수 2: PyMuPDF 미설치
```
ModuleNotFoundError: No module named 'fitz'
```
**해결**: 
```bash
pip install PyMuPDF
```

### ❌ 실수 3: 저장 안 함
파일 수정 후 저장 안 하고 실행
**해결**: Ctrl+S로 저장!
